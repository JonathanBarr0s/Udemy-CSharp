# Exercício - Diferença

Fazer um programa para ler quatro valores inteiros A, B, C e D. A seguir, calcule e mostre a diferença do produto de A e B pelo produto de C e D segundo a fórmula: DIFERENCA = (A * B - C * D).

### Exemplo de Entrada:

```
5
6
7
8
```

### Exemplo de Saída:

```
DIFERENCA = -26
```

### Exemplo de Entrada:

```
5
6
-7
8
```

### Exemplo de Saída:

```
DIFERENCA = 86
```
