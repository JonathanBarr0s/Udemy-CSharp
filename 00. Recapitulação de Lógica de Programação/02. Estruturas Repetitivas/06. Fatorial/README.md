# Exercício - Fatorial

Ler um valor N. Calcular e escrever seu respectivo fatorial. Fatorial de N = N * (N-1) * (N-2) * (N-3) * ... * 1. Lembrando que, por definição, fatorial de 0 é 1.

### Exemplo de Entrada:

```
4
```

### Exemplo de Saída:

```
24
```

### Exemplo de Entrada:

```
1
```

### Exemplo de Saída:

```
1
```

### Exemplo de Entrada:

```
5
```

### Exemplo de Saída:

```
120
```

### Exemplo de Entrada:

```
0
```

### Exemplo de Saída:

```
1
```
