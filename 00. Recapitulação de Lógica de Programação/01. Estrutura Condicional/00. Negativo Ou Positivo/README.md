# Exercício - Positivo e Negativo

Fazer um programa para ler um número inteiro, e depois dizer se este número é negativo ou não.

### Exemplo de Entrada:

```
-10
```

### Exemplo de Saída:

```
NEGATIVO
```

### Exemplo de Entrada:

```
8
```

### Exemplo de Saída:

```
NAO NEGATIVO
```

### Exemplo de Entrada:

```
0
```

### Exemplo de Saída:

```
NAO NEGATIVO
```
